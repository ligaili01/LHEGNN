Prediction of three-dimensional protein-ligand binding affinity represents a critical task in drug discovery. While equivariant graph neural networks have achieved significant progress by encoding 3D symmetries, existing methods still lack hierarchical geometric modeling from local to global scales. To address this limitation, we propose a Local-Hierarchy and Global-Geometry Equivariant Graph Neural Network (LHEGNN).

LHEGNN comprises two core modules: local substructure encoding and cross-local information transfer. The model achieves hierarchical geometric representation by constructing local 3D subtree structures and incorporates a multi-scale feature extraction mechanism. Furthermore, we design an SO(3)-equivariant graph module. It encodes inter-atomic displacements with spherical harmonics for equivariant edge features and uses equivariant message passing. After several layers, each node obtains a final equivariant representation.
